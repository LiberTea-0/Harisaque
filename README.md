# 📝 Personal Blog (FastAPI)

A simple personal blog built with FastAPI for writing down thoughts, ideas, and daily reflections.

This project is designed as a lightweight local blogging system with basic content management features.

---

## 🚀 Features

- Create, edit, and delete posts
- Admin login system
- Tag-based categorization
- Post search functionality
- Clean and minimal reading interface

---

## 🛠 Tech Stack

- FastAPI
- SQLAlchemy
- SQLite
- Jinja2 Templates
- HTML / CSS

---

python -m uvicorn main:app --reload
